# Human vs AI Detection Challenge

An interactive, production-ready single-page web application (SPA) where users play a high-fidelity game to determine whether a given image, text snippet, code block, vocal track, or artwork sample was crafted by a human or generated by an artificial intelligence.

This application is built with a deep dark futuristic theme featuring glassmorphism elements, CSS visual variables, circular animated SVG timer nodes, Canvas particle fields, and integrated Chart.js performance profiling boards.

## Features

1. **Category Modules**:
   - **Visual Image**: Evaluate vector shapes (SVGs) looking for perfect line snaps vs AI alignment overlaps.
   - **Written Text**: Detect rigid, highly structured transition flows and repetitive structures.
   - **Program Code**: Evaluate clever micro-hacks and informal inline comments against textbooks documentation layouts in Javascript, Python, C++, Java, HTML, and CSS.
   - **Vocal Speech**: Listen to browser spoken transcripts using Web Speech Synthesis and look at simulated voice waveforms.
   - **Fine Artwork**: Evaluate symmetrical radial compositions against layered brush anomalies.
   - **Mixed Protocol**: Runs a dynamic sequence incorporating all categories randomly.

2. **Advanced Score Matrix**:
   - Dynamic Multipliers based on chosen threat level: Easy (1.0x), Medium (1.5x), Hard (2.0x), Expert (3.0x).
   - Fast-Answer bonuses for rapid answers.
   - Streak bonuses (+20 points for every 5 consecutive correct answers).
   - Penalty tracking (hints reduce potential bonuses).

3. **Database Ledger & Statistics**:
   - Top 20 Global Leaderboards persisted on LocalStorage with search filters and sorting headers.
   - Interactive profile statistics rendering radar performance profiles, difficulty bar distributions, and category share dough-charts.
   - Reusable Confirmation Dialogs to reset progress, wipe records, or erase configuration sets.

4. **Web Synthesizers & Voice System**:
   - **Web Audio API Sound Engine**: Generates sound effects (chimes, clicks, countdown warning frequencies, victory rolls, sad game-over loops) and soft cyber ambient background drones programmatically. Works 100% offline.
   - **Web Speech Synthesis API**: Speeds up, pitches down, or adjusts vocal cadences to simulate mechanical robot voices vs. natural human pacing.

5. **Accessibility Suite**:
   - High-contrast visual toggles.
   - Prefers-Reduced-Motion media listener support.
   - Keyboard Tab outlining indicators to prevent outline clutter on mouse clicks.
   - ARIA live region screen reader live text vocalizer.

## Installation & How to Run

Since this application is written entirely using vanilla ES6 modules, it does not require complex compilers or external dev environments. However, due to browser security restrictions on loading ES6 modules from local files (`file://` protocol), it **must be served via a local web server**.

### Option 1: Live Server (VS Code)
1. Open the project root folder in VS Code.
2. Install the **Live Server** extension.
3. Click the **Go Live** button in the status bar at the bottom right.

### Option 2: Node.js (http-server)
1. Ensure Node.js is installed.
2. Open terminal and run:
   ```bash
   npx http-server
   ```
3. Open `http://127.0.0.1:8080` in your web browser.

## Technologies Used

- **Markup & Styling**: HTML5, CSS3 Custom Properties (Variables)
- **Programming Layer**: Vanilla JS (ES6 Modules, Classes, Web Speech API)
- **Audio Routing**: Web Audio API (Dynamic Sine, Triangle, and Sawtooth Oscillators)
- **Analytics Canvas**: Chart.js (Radar, Bar, Doughnut charts)
- **Icon Assets**: FontAwesome CDN
