# ExpenseTrack - Modern Personal Finance Tracker

A complete, modern, responsive Personal Finance Tracker web application built entirely with HTML, CSS, and Vanilla JavaScript. The project features a premium glassmorphism design, dark/light mode, local storage persistence, and beautiful dynamic charts.

## Features

- **Dashboard**: Get a quick overview of your total expenses, number of transactions, and highest spending category for the current month.
- **Dynamic Visualizations**: View your expenses broken down by category in a Pie Chart, and track your monthly spending trend in a Bar Chart (powered by Chart.js).
- **Expense Management**: Add, edit, and delete expenses easily. Each expense includes a date, description, category, and amount.
- **Budget Limits**: Set monthly budgets for specific categories. See visual progress bars indicating how close you are to your limits, and get alert warnings when you exceed them.
- **Filtering & Sorting**: Easily search through your expenses, filter by category or month, and sort by date or amount.
- **Local Storage**: All your expenses and budget data are saved in your browser's Local Storage, meaning your data stays safe even if you refresh or close the page.
- **Theme Toggle**: Switch seamlessly between a clean Light mode and a sleek Dark mode.
- **Responsive Design**: The application works perfectly on desktop, tablet, and mobile devices with a custom off-canvas sidebar on smaller screens.
- **Interactive UI**: Enjoy smooth hover effects, micro-animations, and toast notifications for every action.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Variables, Flexbox, Grid, Glassmorphism, Animations.
- **Vanilla JavaScript (ES6)**: DOM Manipulation, State Management, LocalStorage, Array Methods.
- **Chart.js**: For rendering the Pie and Bar charts.
- **Font Awesome**: For UI icons.
- **Google Fonts**: 'Outfit' for typography.

## Setup Instructions

This project requires no build tools or server setup. 

1. Clone or download this repository.
2. Open the project folder.
3. Double-click on `index.html` to open it in any modern web browser.
4. Start tracking your expenses!
